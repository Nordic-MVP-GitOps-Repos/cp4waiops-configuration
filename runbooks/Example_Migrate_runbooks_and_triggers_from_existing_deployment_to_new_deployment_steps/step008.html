Run the command

curl -X POST -k -s -u "" -H "Expect:" -H "Content-Type: application/json" -d@./exportedRunbooks.json "https:///api/v1/rba/runbooks/import?publish=true" > importedRunbooks.json
  • once for the single file exportedRunbooks.json (in case its size is below 1 MB)
  • or multiple times, once for each of the chunks exportedRunbooks.json.* that you have created in the previous step. In this case, you should also modify the name of "importedRunbooks.json" to a file name that corresponds to the name of the chunk, so each pair of export and import files can be compared later on.
