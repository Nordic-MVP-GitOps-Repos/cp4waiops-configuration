Check whether the ea-actions topic is populated:

oc exec -ti -kafka-0 -- ../../opt/kafka/bin/kafka-console-consumer.sh --from-beginning --bootstrap-server localhost:9092 --topic ea-actions

If no entries are present check the inference service logs:

oc logs $(kubectl get pod | grep inference | awk '{print $1}')

Restart the service, if required:

oc delete pod $(kubectl get pod | grep inference | awk '{print $1}')
