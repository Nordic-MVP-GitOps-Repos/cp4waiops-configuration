Run the following command to display the ten processes consuming the most memory:

 

ps -eo %mem,pid,comm | sort -nr | head -n 10

 

The result will display the memory utilization in the first column. Identify all processes with a too high memory utilization, go back to your runbook parameters and change the PROCESS_ID parameter to the number identified by the second column.

If there are no processes appearing in this list with a high memory utilization, this runbook.
