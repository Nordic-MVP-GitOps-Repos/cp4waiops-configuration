If you have reached this step of the runbook, the process is most likely out of control and needs to be stopped by force. Any data held in memory or data not yet written will be lost. Run the following command to force the process to stop:

 

kill -KILL

 

Wait for a few seconds, then run the following command to check that the process has ended accordingly:

 

ps -ef | grep | grep -v grep

 

Repeat this step for the remaining processes identified. If the output for all processes is empty the runbook. If at least one process is still running continue with the next step.
