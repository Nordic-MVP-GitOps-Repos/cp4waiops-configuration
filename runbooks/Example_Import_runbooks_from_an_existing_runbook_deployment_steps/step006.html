Check (and if necessary, update) the value of the SOURCE_SINGLERUNBOOKID parameter.

  • To export all runbooks at once, leave SOURCE_SINGLERUNBOOKID empty
  • To export a single runbook, set SOURCE_SINGLERUNBOOKID to the slash ("/") character followed by the runbook id, for example, /397ec21f-0d4c-4e1e-8652-d5c8a15f5371
  • If you have modified the parameter value click "Update parameters".
