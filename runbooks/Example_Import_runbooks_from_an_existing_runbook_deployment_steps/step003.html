Instructions for IBM Cloud Pak for Watson AIOps as the target

  • Set the value of the TARGET_BASEURL parameter to the IBM Cloud Pak for Watson AIOps user interface URL and replace the path with "/aiops/api/story-manager/rba/v1". For example, if the URL is "https://cpd-cp4waiops.apps.mycluster.mycompany.com/zen/#/homepage", then set the TARGET_BASEURL to https://cpd-cp4waiops.apps.mycluster.mycompany.com/aiops/api/story-manager/rba/v1
  • Once you have set the value click "Update parameters".

 

Instructions for IBM Netcool Operations Insight running on Red Hat OpenShift as the target

  • Set the value of the TARGET_BASEURL parameter to the host name where the IBM Netcool Operations Insight service is running, prefix it with "https://" and append "/api/v1/rba", for example, https://netcool.n163.apps.mycluster.mycompany.com/api/v1/rba
  • Once you have set the value click "Update parameters".

 

Instructions for IBM Runbook Automation Private Deployment as the target

  • Set the value of the TARGET_BASEURL parameter to the host name and port where the runbook service is running (which may be different from the location where the Netcool OMNIbus/DASH UI server is running). The value has the format "https://<fullyQualifiedDomainName>:<rba-rbs-port>/api/v1/rba", for example, https://myhost.mycompany.com:3005/api/v1/rba
  • Once you have set the value click "Update parameters".
