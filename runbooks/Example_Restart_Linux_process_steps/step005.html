Run the command below to check whether the process is now running:

 

ps -eo args | grep -E "( .*|$)" | grep -v grep

 

If the result returned contains a line with the processname plus any optional additional information then the process is running. If the command didn't return a result the process is not yet running. Your application might need more time to start. Wait for one minute, then repeat the check. If the process is not running after 10 minutes, open a ticket.
