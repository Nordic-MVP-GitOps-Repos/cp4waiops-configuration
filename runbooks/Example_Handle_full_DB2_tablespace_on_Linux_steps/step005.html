Run the following command:

 

db2 list tablespace containers for show detail

 

The result should display the container ID, name, and type, similar to the example below:

db2inst1@db2server:~> db2 list tablespace containers for 7
            Tablespace Containers for Tablespace 7
Container ID                         = 7
Name                                 = /home/db2inst1/db2inst1/NODE0000/DB2SERVER/T0000000/C0000000.CAT
Type                                 = File
Total pages                          = 1028
Useable pages                        = 1016
Accessible                           = Yes

It also provides the information that there are 1,016 usable pages. Later in this runbook the tablespace is increased by 10%. In the example this would be roughly 100 pages. Go back to runbook details and add that number as the value of runbook parameter PAGES_TO_ADD. Also add the name of the filesystem, e.g. /home, as value for runbook parameter FILESYSTEM. Press apply then and proceed with the next step.
