The API endpoint "DELETE /.../runbookinstances/:runbookInstanceId" allows you to delete the execution data.

For every runbook instance successfully extracted in step 5, run the following command. Again replacing the term "RID" with your runbookInstanceId:

 

runbookInstanceId=RID;curl -X DELETE -k  "/runbookinstances/${runbookInstanceId}?cascade=true"
