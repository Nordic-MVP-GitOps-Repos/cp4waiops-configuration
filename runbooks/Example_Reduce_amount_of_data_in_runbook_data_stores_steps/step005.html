The API endpoint "GET /.../runbookinstances" allows you to extract the execution data in various ways. Before getting the data, you should determine the timeframe of the data to be extracted. You need a start date and an end date. If you leave the START_DATE parameter empty then the start date defaults to "7 days ago". If you leave the END_DATE parameter empty then the end date defaults to "today". All data formats which can be parsed by moment.js are supported. The current start date is . Gather the list of runbook instance IDs by running the following command and store the output:

 

curl -X GET -k  "/runbookinstances?from=&to=&fields=_id"
